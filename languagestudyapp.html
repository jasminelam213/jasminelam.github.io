Language Study

Study languages with flashcards and virtual calls.

Studying different languages can be difficult. Languge Study dedicates it's app to be more colorful and fun. There are multiple features such as flashcards, fill in the blanks, and a video call to recieve help on pronunciation.

But, having these features came with multiple challenges.

Overview

During the summer of 2021, I was a scholar at Google CSSI where we learned about different mark-up languages such as HTML, CSS, and Javascript. Towards the end of the program, I had to create a website witha with a team of two, creating a language study app. Although, we knew about the multiple language study applications, we wanted to create our own features. The problem arose when working on the different features of the language study apps.

Problem

  • The color themes included are typically used for older audiences.
  • Did not include accessible video call features.

Problem Solution

  • Gave the users an option for pink or blue backgrounds, to make it more enjoyable.
  • Add a call function for users to communicate with other to practice languages.
