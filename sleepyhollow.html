Sleepy Hollow

Keep track of sleep and have fun!

With the rise of social media, kids are losing their recommended amount of sleep. To make their experience more fun, Sleepy Hollow is a website that allows kids to grow a plant everytime they sleep for the recommended amount of time.

But, building this website came with multiple challenges.

Roles

User Experience (UX) Designer

User Interface (UI) Designer

Software Developer

Project Specifications

Duration: 2 Days

Tools

  • VSCode
  • Figma

Overview

During the summer of 2020, I participated in a Hackathon at Google that utilized HTML, CSS, and Javascript. Towards the end of the program, I had to create a website witha with a team of three, creating a wellness and heath app. We wanted to create a website that is dedicated towards children. The problem arose when working on the different different components needed for a children app.

Problem

  • Ensure that children would use this application.
  • Esnure that the app is enjoyable.

Problem Solution

  • Include images of plants that children can grow when they sleep the recommended amount of sleep.
  • Creative features that makes the app enjoyable.
